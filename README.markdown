Simple Log By MongoDB!
Just use for my project!

**need rails3+/mongoid2.0+**

Usage:

  example:
  
  rails g mongoid:config
  
  gem 'log_fu'
  
  1. rails g log_fu:install
  2. rails g log_fu:mongo_model subject_log
  
  *Only support MongoDB with mongoid*
